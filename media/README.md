# Media Directory

This directory contains static media files for your portfolio website.

## CV File

Place your CV file here with the following name:
- `Casper_Moyo_CV.pdf`

The download CV functionality will automatically look for this file when users click the "Download CV" button.

## Supported Formats

- PDF (recommended)
- DOC/DOCX
- TXT

Make sure your CV file is named exactly `Casper_Moyo_CV.pdf` for the download to work properly.
